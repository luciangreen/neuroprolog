# Neurocode

This directory contains neurocode — optimised Prolog generated by the NeuroProlog compiler pipeline.

## Invariants

All files in this directory must:

- Be valid Prolog.
- Be loadable into a standard Prolog system without modification.
- Be diffable in Git.
- Not contain binary, byte-level, or opaque representations.
- Be traceable to the original source via cognitive code markers.

## Generating Neurocode

```sh
swipl -g "consult('src/neuroprolog')" \
      -g "npl_compile('src/neuroprolog.pl','neurocode/neuroprolog_nc.pl')" \
      -t halt
```

## Self-Compilation

```sh
swipl -g "consult('src/self_host')" -g "self_compile" -t halt
```
